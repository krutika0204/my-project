# 🏥 Hospital Data Analysis using SQL

## 📌 Project Overview
This project analyzes hospital patient data using SQL to extract meaningful insights like treatment cost, patient conditions, readmission rate, and satisfaction.

---

## 🗂️ Database
- MySQL
- Database Name: HospitalDB

---

## 📊 Dataset Columns
- Patient_ID
- Age
- Gender
- Condition
- Procedure
- Cost
- Length_of_Stay
- Readmission
- Outcome
- Satisfaction

---

## ⚙️ Features & Analysis

### 🔹 Basic Queries
- View all records
- Select specific columns

### 🔹 Aggregation
- Patient count by condition
- Total and average cost
- Average length of stay

### 🔹 Advanced SQL
- Readmission rate calculation
- Subqueries (above average cost/stay)
- Group-based analysis

### 🔹 Joins
- Created:
  - Patient_Info table
  - Treatment_Info table
- Performed JOIN using Patient_ID

---

## 📈 Insights
- Identified costly treatments
- Analyzed readmission trends
- Compared satisfaction scores
- Found condition-wise patient distribution

---

## 🚀 How to Run
1. Run `hospital_schema.sql`
2. Insert data into table
3. Run `hospital_queries.sql`
